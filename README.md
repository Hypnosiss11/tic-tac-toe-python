# Tic Tac Toe (Python)

A simple console-based Tic Tac Toe game written in Python.  
Players can enter their moves by selecting the row and column.  
The game checks for invalid moves, turn errors, and determines the winner  
(horizontal, vertical, or diagonal). If the board is full and there is no winner, the game ends in a draw.

## Features
- 3x3 matrix game board
- Player input validation
- Prevents moves on already occupied cells
- Detects winning combinations (row, column, diagonal)
- Detects draw when the board is full

## How to Run
Make sure you have Python 3 installed.  
Then run:

```bash
python tic-tac-toe.py
